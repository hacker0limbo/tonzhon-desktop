## 开发

```bash
npm install
npm run start
```

如果安装失败, 请尝试使用国内镜像, 可参考官方文档的[镜像部分](https://www.electronjs.org/zh/docs/latest/tutorial/installation#%E9%95%9C%E5%83%8F)

在 `.npmrc` 文件中添加如下内容:

```bash
electron_mirror=https://npmmirror.com/mirrors/electron/
```
